Add Multiple Students

Class list must contain a list of the students, one student per line. Each line can have any of the following three formats:

A single email address. (Example: jdoe@example.com)

A standard CSV style with last name, first name and email. (Example:Doe, John, jdoe@example.com)

Last name, then a comma, then first name, followed by the email in parenthesis. (Example: Doe, John (jdoe@example.com)).

(Note: A full class list in the above format can be generated from Banner by going to "Summary Class List w/o Hyperlinks", selecting the top-most checkbox under the E-Mail column, then clicking on "Send email to selected students".)
